# MultiPDFPrinter
Programme d'impression PDF avec mode livret.

## 1. Installation rapide
```bash
git clone https://github.com/vuillaumetheo10-maker/MultiPDFPrinter.git
```

### 2. Installation des dépendances Python 
```bash
pip install -r requirements.txt
```

###3. Lancement du programme :


```bash
python multi_pdf_printer.py
```

Sinon :

Possibilité de créer un raccourcis sur le bureau avec un clique droit "Nouveau" + "Raccourci" Rentrer : python "C:\chemin\vers\ton\dossier\fichier.py"

###4. Installation de SumatraPDF

L'installation de Sumatra se propose automatiquement lors de l'utilisation du logiciel en cliquant sur "TrouverSumatraPDF"

sinon :

Télécharger : https://www.sumatrapdfreader.org/download

Dépendances

    pypdf : Manipulation des PDF

    pywin32 : Gestion des imprimantes Windows

    SumatraPDF : Moteur d'impression
